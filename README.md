# CX AI Knowledge Base

基于 DeepSeek 大模型 + RAG 技术的企业级 AI 知识库问答系统。

## 技术栈

- 后端：FastAPI + DeepSeek API + sentence-transformers
- 前端：Vue 3 + Element Plus

## 快速启动

### 后端

    pip install -r requirements.txt
    python run.py

### 前端

    cd frontend
    npm install
    npm run dev

访问 http://localhost:5173 使用系统。

## 功能

- 文档上传（PDF/DOCX/TXT）
- 语义检索问答
- 参考来源溯源
